# Mini ISS Map

Tento projekt zobrazuje aktuální polohu Mezinárodní vesmírné stanice (ISS) na mapě v reálném čase. Používá knihovnu Leaflet pro zobrazení mapy a API pro získání polohy ISS.

## Jak používat

1. Otevři soubor `index.html` ve svém prohlížeči.
2. Mapa se automaticky aktualizuje každých 5 sekund a ukáže polohu ISS.
